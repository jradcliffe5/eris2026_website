AT1 - Metre / wide-field (LOFAR)

← back to top

  1. Install ds9 for data inspection
    Download following the instructions here:
    https://sites.google.com/cfa.harvard.edu/saoimageds9/download


  2. Install a working container application, Apptainer or Docker:
    • Apptainer instructions: https://apptainer.org/docs/admin/main/installation.html
    • Docker instructions: https://docs.docker.com/desktop/install/mac-install/

    Note: If you encounter issues having a working Apptainer environment on your machine (e.g. if you use MacOS with a non-Intel processor), use the docker option above which is the recommended option.


  3. Obtain the needed containers
      3.1 Apptainer
      • Download the container from the FLoCs repo to run the facetselfcal.py, available in https://tikk3r.github.io/flocs/
        • Check your processor unit type and use version 5.1.0.
        • Check that the Apptainer & FLoC installed properly, e.g.: 
          $ apptainer shell flocs_v5.1.0_sandybridge_sandybridge_mkl_cuda.sif
      • Pull the LINC container (it will take a few minutes) 
        $ apptainer pull linc:v4.0-89.sif docker://astronrd/linc:v4.0-89-ge1f9b67

      Note: To run the FLoCs containers, you need to install Apptainer, not Singularity

      3.2 Docker
      • Download the docker file, available in https://git.astron.nl/loose/eris/-/blob/main/eris2024/Dockerfile?ref_type=heads
      • For Linux / Mac users with Docker installed (and started - MAC users), with an opened terminal window: 
        $ mkdir TMP_ERIS2024
$ cp /<path to the downloaded>/eris2024at1.def TMP_ERIS2024
$ cd TMP_ERIS2024
$ docker build -t eris2024_soft -f eris2024at1.def .

  4. Download data & ancillary files
    From the data repository: https://filesender.surf.nl/?s=download&token=3a59b45d-30f9-4392-932c-07a1ce2b3202

    Untar the content and move it to your favourite path

    $ tar -xvf ERIS2024_AT1.tar
$ mv ERIS2024_AT1 /<to your path>

AT4 - Millimetre / spectral line (ALMA)

← back to top

For the Millimetre/spectral line tutorial we will process ALMA data in CASA.

Software Requirements:

  • Firefox is required to visualize ALMA weblogs. Alternatively, except for participants with Mac ARM, it is possible to visualize weblogs using CASA directly without the need of Firefox. (more details here)
  • The same CASA version that has been installed for the rest of the school. Unfortunately, participants with Mac ARM will not be able to run the ALMA Science Pipeline with that CASA version on their laptops. Information on how to overcome this limitation will be updated before the tutorial. In the next few days (hopefully before our tutorial day), a new version of the ALMA Pipeline will be released, packed with CASA 6.6.1. That should work on Mac ARM (TBC).
  • The same CARTA version 4.1.0 will be used for the data analysis part of the tutorial.
